Function Concepts in AI Development

1. Basic Function Structure

Key Concept

Functions are reusable blocks of code that perform specific tasks. In AI, they're used to organize data processing, model training, and prediction logic.

# Basic function structure def preprocess_data(data, normalize=True): """ Preprocess input data for machine learning model Parameters: data: Input features normalize: Whether to normalize the data Returns: Processed data ready for model """ if normalize: # Simulate normalization processed_data = data / 100 else: processed_data = data return processed_data # Example usage sample_data = [150, 200, 300] processed = preprocess_data(sample_data) print(f"Original data: {sample_data}") print(f"Processed data: {processed}")

2. AI-Specific Function Examples

Real-world Application

Let's create a simple feature scaling function commonly used in machine learning:

def min_max_scale(features): """ Scale features to range [0,1] using min-max scaling Parameters: features: List of numerical values Returns: Scaled features """ min_val = min(features) max_val = max(features) scaled = [(x - min_val) / (max_val - min_val) for x in features] return scaled # Test the function raw_features = [10, 20, 30, 40, 50] scaled_features = min_max_scale(raw_features) print(f"Original features: {raw_features}") print(f"Scaled features: {scaled_features}")
